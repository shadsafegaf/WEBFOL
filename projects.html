CHESS LIBRARY

Web Application

A web-based Chess Library Management System that streamlines book cataloging, borrowing, returning, and penalty tracking. The system features a responsive React.js front-end, Node.js/Express.js back-end, and a secure MySQL database.

Key Features

  • Real-time borrowing/return tracking with automatic due dates and fines
  • CRUD book management with advanced search and filters
  • Role-based access for Admins and Members
  • Transaction and stock reports generation
  • Responsive interface for desktop, tablet, and mobile
